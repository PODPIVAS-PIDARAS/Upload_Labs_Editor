This is save editor for Upload Labs. 
U can Edit core values: Modify corporation_data, government_data, money, research, research_advanced, token, and code_level

Preset system:
A default preset sets all values (except code_level) to 2.080210911784015e+60.
Save your own custom presets and apply them with one click.
Presets are stored in presets.json next to the executable.

Limit protection:
code_level is always capped at 300.
Optional per‑variable limits with visual validation (red background for invalid numbers).
Backup: Automatically creates a .bak copy of your save before writing changes.
Dark/Light theme: Toggle between light and dark mode for comfortable viewing.

How to Use:
1.Download the latest UploadLabsEditor.exe
2.Run the executable.
3.Select your save file (default path is automatically suggested).
4.Click Load to read current values.
5.Edit the fields as needed.
6.Optionally enable limit checks and set maximum values.
7.Click Save – a backup will be created and your changes applied.

CRITICAL: When changing saves, the game must be turned off
